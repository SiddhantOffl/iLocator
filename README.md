### Features

- Beginners friendly
- Kali Linux `Upadate & Upgrade` 
- Requirements Installing 
- Tools Install `Basic Hacking Tools`

### Installation

- Just, Clone this repository -
```
$ git clone https://github.com/SiddhantOffl/iLocator.git
```

- Change to cloned directory and run `iLocator.sh` -
```
$ cd basic
$ bash iLocator.sh
```

- On first launch, It'll install the dependencies and that's it. `iLocator.sh` is installed.
